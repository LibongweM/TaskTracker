# Task Tracker

Task Tracker system that a team could use to manage their work

## Local Development

### Requirements

-   NodeJS >= v20.19.0
-   Angular Common v19.2.0
-   .NET SDK Runtime: v8
-   AspNetCore Runtime: v8
-   C# compatible IDE: Visual Studio, JetBrains Rider, Visual Studio Code + C# Dev Kit
-   Angular compatible IDE: Visual Studio Code, WebStorm, Cursor

### Running Locally

#### Web API

Navigate to the location of the `TaskTracker.csproj` inside the folder structure `<repo-name>/Backend/TaskTracker` then you can execute the following in your command line:

```bash
dotnet run
```

The application will start up on port(s): `http://localhost:5000` or `https://localhost:5001`.

#### Angular SPA

Navigate to the location of the `package.json` inside the folder structure `<repo-name>/task-tracker-fe/TaskTracker` then you can execute the following in your command line:

```bash
npm run start
```
